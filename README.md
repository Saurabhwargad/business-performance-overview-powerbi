# Business Performance Overview - Power BI Dashboard

## Project Overview
This dashboard provides a comprehensive view of business performance, analyzing sales, profit, shipping cost, product performance, and regional trends.

## Key Analysis Areas
- Sales & Profit by Region
- Sales by Product
- Profit by Segment (Consumer, Corporate, Home Office)
- Shipping Cost by Ship Mode
- Year-wise Sales Distribution
- Order Priority Analysis

## Key Insights
- Consumer segment contributes highest profit.
- Standard Class shipping incurs highest shipping cost.
- Central and South regions show strong sales contribution.
- Top products drive majority of revenue concentration.

## Tools Used
- Power BI
- DAX Measures
- Data Modeling
- Business KPI Analysis

## Objective
To transform raw transactional data into strategic insights supporting business performance evaluation and decision-making.
